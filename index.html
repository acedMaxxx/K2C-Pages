IP DATA SHEET: K2C BF16-TO-INT4 ENGINE

VERSION 1.0.0 | RELEASE 2025 | STATUS: SILICON-READY

DESCRIPTION

K2C is a high-throughput quantization core designed for LLM KV-cache acceleration. The IP transforms 512-bit Brain-Float 16 vectors into 144-bit quantized structures in a single clock cycle, enabling a 4x reduction in off-chip memory bandwidth requirements.
